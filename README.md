# TiCloudRTC iOS SDK

TiCloudRTC 是一个基于 Agora RTC/RTM 的 iOS 实时语音呼叫 SDK。
当前仓库提供 `3.10.5` 的 CocoaPods 二进制发布内容。

## Installation

```ruby
pod 'TiCloudRTC', '3.10.5'
```

## Requirements

- iOS 12.0+
- Xcode 16 或更高版本
- 仅支持 iOS Device `arm64`

## Dependencies

- `AgoraRtm_iOS` `1.5.1`
- `AgoraRtcEngine_iOS` `4.3.2`

## Notes

- 当前发布产物为 `TiCloudRTC.framework`
- 不包含 iOS Simulator slice
